# StudyVault AI - Figma MVP Design Prompt

## 🎯 Project Overview

Create a comprehensive Figma design for StudyVault AI MVP - an intelligent study assistant application featuring Material Design 3, glass morphism effects, and AI-powered study optimization. This design should showcase the complete user journey with all planned features and algorithms.

## 🎨 Design System Requirements

### Material Design 3 + Glass Morphism Hybrid

```
Core Design Principles:
- MD3 Foundation: Color tokens, typography, elevation, shapes
- Glass Morphism: Frosted glass effects, blur backgrounds, transparency
- Depth & Layering: Multiple elevation levels with glass surfaces
- Modern Aesthetics: Clean, sophisticated, professional
- Accessibility: WCAG AA+ contrast ratios
```

### Color Palette

```
Primary System:
- Primary: #d0bcff (Light Purple)
- On Primary: #001d36 (Dark Blue)
- Primary Container: #004494 (Deep Blue)
- On Primary Container: #cce7ff (Light Blue)

Glass Morphism Layers:
- Surface Glass: rgba(13, 17, 23, 0.7) - Dark glass base
- Card Glass: rgba(25, 28, 28, 0.8) - Frosted card effect
- Accent Glass: rgba(208, 188, 255, 0.2) - Purple accent glass
- Border Glass: rgba(255, 255, 255, 0.1) - Subtle borders

Background System:
- Base: #0d1117 (GitHub dark)
- Gradient: radial-gradient(circle at 20% 50%, #1a1f2e 0%, #0d1117 100%)
- Overlay: Subtle noise texture for depth
```

### Typography System

```
Display Scale:
- Display Large: 57px - Hero sections
- Display Medium: 45px - Page titles
- Display Small: 36px - Section headers

Body Scale:
- Headline Medium: 32px - Card titles
- Title Medium: 22px - Subsection titles
- Body Medium: 14px - Main content
- Label Medium: 12px - Metadata, buttons

Font Stack:
- Display: 'Plus Jakarta Sans' - Modern, geometric
- Body: 'Inter' - Clean, readable
- Code: 'JetBrains Mono' - Technical elements
```

### Glass Morphism Effects

```
Visual Hierarchy:
- Level 0: Base background with gradient
- Level 1: Frosted glass surfaces (blur 12px)
- Level 2: Elevated glass cards (blur 16px + shadow)
- Level 3: Interactive elements (blur 20px + glow)

Effects Library:
- Frosted Glass: backdrop-filter: blur(12px) + rgba background
- Glass Borders: 1px solid rgba(255, 255, 255, 0.1)
- Glass Shadows: 0 8px 32px rgba(0, 0, 0, 0.3)
- Glass Glow: 0 0 20px rgba(208, 188, 255, 0.3)
```
